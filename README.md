Common classes and helpers for BBK library stack.

## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec

Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "bbk-utils", "~> 1.0.0"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile

gem "bbk-utils", "~> 1.0.0"
```

### Supported Ruby versions

* Ruby (MRI) >= 2.5.0
